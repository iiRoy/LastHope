## 1. What the Project Is

**Last Hope** is a terminal-based Python game in which the player takes command of humanity’s last military fleet and must destroy enemy ships before running out of projectiles. The game is inspired by the logic of **Battleship**, but it is adapted into a science-fiction setting with original narrative framing.

The project includes:
- An **Arcade Mode** with multiple difficulty levels
- A **Story Mode** with narrative scenes
- A **10x10 board system**
- **Random enemy ship placement**
- **Hit and miss detection**
- **Victory and defeat conditions**
- A hidden **developer/debug menu** for gameplay testing

## 3. Setup Instructions

### Requirements
- Python 3 installed on your system

### Run the Project
1. Download or clone the project files.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python LastHope.py
```

> [!NOTE] 
> The game runs entirely in the terminal. No external dependencies are required. Player input is coordinate-based, using values such as A3.

## 5. A Brief Explanation of What I Learned or Solved

This project helped me develop a stronger understanding of how to build an interactive game using Python and terminal-based logic. I worked on structuring a complete gameplay loop, validating player input, and representing a board system through a two-dimensional matrix.

Some of the main challenges I solved were:

- Building a playable targeting system on a 10x10 board
- Randomly placing ships without overlap
- Managing game states such as hits, misses, remaining projectiles, and victory conditions
- Designing different gameplay modes, including Arcade and Story
- Integrating a narrative layer into a logic-based game
- Creating internal debug options for testing and adjusting gameplay behavior

Overall, this project gave me practical experience transforming a classic board game concept into a themed digital experience with structure, progression, and replayability.
